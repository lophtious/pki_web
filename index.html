OpenSSL Command

Run this command in your terminal to generate an identical key and CSR.

openssl req -new -newkey rsa:2048 -nodes -keyout private.key -out request.csr -subj "/CN=mtls.company.com/O=Contoso, LLC/OU=Security Operations/C=US/L=Delaware/ST=California/emailAddress=infosec@company.com"
